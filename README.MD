✈️ Airline Delay Performance Analysis Dashboard
📊 Project Overview

The Airline Delay Performance Dashboard is an interactive data visualization project built using Tableau Public.
This project analyzes airline operational performance by examining flight delays, delay reasons, airline comparisons, and aircraft emissions.

The dashboard provides valuable insights that help understand patterns, causes of delays, and airline efficiency over time.

The dataset contains flight operational data including:

Airline name

Flight dates

Delay times

Delay reasons

Aircraft types

Flight cancellations and diversions

🎯 Project Objective

The objective of this project is to analyze airline delay performance and visualize operational insights using interactive dashboards for better decision-making.
